Chapter 5: Discrete Probability Distributions

Binomial Distribution

  • When: n independent trials, two outcomes, constant p
  • Formula: P(X=x) = C(n,x) × px × (1-p)(n-x)
  • Mean: μ = np, Variance: σ² = np(1-p)

Poisson Distribution ⭐⭐

  • When: Rare events, constant rate λ, independent
  • Formula: P(X=x) = (e-λ × λx) / x!
  • Critical: Mean = Variance = λ

Other Discrete Distributions

  • Geometric: Trials until first success
  • Negative Binomial: Trials until rth success
  • Hypergeometric: Sampling without replacement (NOT binomial)

Chapter 6: Continuous Probability Distributions

Normal Distribution ⭐⭐⭐

  • MUST MEMORIZE the PDF formula
  • Properties: Bell-shaped, symmetric, inflection points at μ±σ
  • Empirical Rule: 68% within μ±σ, 95% within μ±2σ, 99.7% within μ±3σ
  • Z-score: Z = (X - μ)/σ
  • Tables will be provided (Z-tables)

Exponential Distribution

  • When: Time until first event, waiting times
  • Formula: f(x) = λe-λx
  • Memoryless Property: P(X > s+t | X > s) = P(X > t)
  • Mean: μ = 1/λ, Variance: σ² = 1/λ²

Weibull Distribution ⭐

  • MUST MEMORIZE formulas
  • When: Failure/reliability analysis, lifetime data
  • Extremely flexible (includes exponential as special case)
  • Parameters: α (shape), β (scale)

Other Continuous Distributions

  • Gamma: Time until kth event
  • Chi-square: Variance testing, goodness-of-fit
  • Beta: Proportions on [0,1] interval
  • Lognormal: Right-skewed positive data
  • Uniform: Equal likelihood over interval

Formulas for gamma, beta, lognormal, and chi-square will be provided

Chapter 11: Simple Linear Regression (Sections 11.1-11.3)

Linear Regression Model

  • Model: Y = β₀ + β₁X + ε
  • β₀: Y-intercept, β₁: Slope
  • Assumptions: Independent errors, normally distributed, constant variance

Least Squares Estimation

  • b₁ = [Σxy - n(x̄)(ȳ)] / [Σx² - n(x̄)²]
  • b₀ = ȳ - b₁x̄
  • Fitted line: ŷ = b₀ + b₁x

Interpretation

  • Slope: Change in Y per unit change in X
  • Intercept: Y value when X = 0
  • Residuals: eᵢ = yᵢ - ŷᵢ (actual - predicted)

🎯 Probability Distributions Quick Reference

13 Major Distributions You Need to Know

1️⃣ BINOMIAL DISTRIBUTION

When: Fixed n trials, 2 outcomes per trial, constant p, want number of successes

Formula: P(X = x) = C(n,x) × px × (1-p)(n-x)

Mean & Variance: μ = np, σ² = np(1-p)

Scenarios: Quality control, survey sampling, yes/no outcomes

Must Memorize? YES

2️⃣ POISSON DISTRIBUTION ⭐⭐

When: Events occur independently at constant rate λ, rare events

Formula: P(X = x) = (e-λ × λx) / x!

CRITICAL: μ = σ² = λ (mean equals variance!)

Scenarios: Defects per unit, arrivals per time period, rare events

Must Memorize? YES - CRITICAL

3️⃣ GEOMETRIC DISTRIBUTION

When: Number of trials until first success

Formula: P(X = x) = (1-p)(x-1) × p

Mean: μ = 1/p

Must Memorize? Basic understanding

4️⃣ NEGATIVE BINOMIAL DISTRIBUTION

When: Number of trials until rth success

Formula: P(X = x) = C(x-1, r-1) × pr × (1-p)(x-r)

Mean: μ = r/p

Must Memorize? Framework understanding

5️⃣ HYPERGEOMETRIC DISTRIBUTION

When: Sampling WITHOUT replacement from finite population

Formula: P(X = x) = [C(K,x) × C(N-K, n-x)] / C(N, n)

KEY DISTINCTION: Use when binomial assumption violated (no replacement)

Must Memorize? When to use it (CRITICAL!)

6️⃣ NORMAL DISTRIBUTION ⭐⭐⭐

When: Bell-shaped continuous data, foundation of statistics

Formula: f(x) = (1/(σ√(2π))) × exp(-(x-μ)²/(2σ²))

Standardization: Z = (X - μ)/σ

Empirical Rule: 68-95-99.7 rule

Must Memorize? YES - ABSOLUTELY CRITICAL

7️⃣ EXPONENTIAL DISTRIBUTION

When: Time until first event, waiting times, lifetime (no wear)

Formula: f(x) = λe-λx

Mean: μ = 1/λ, Variance: σ² = 1/λ²

CRITICAL PROPERTY: Memoryless - P(X > s+t | X > s) = P(X > t)

Must Memorize? YES

8️⃣ GAMMA DISTRIBUTION

When: Time until kth event, sum of k exponentials

Mean: μ = αβ, Variance: σ² = αβ²

Special Cases: α=1 → Exponential; α=k/2, β=2 → Chi-square

Formula will be PROVIDED

9️⃣ CHI-SQUARE DISTRIBUTION

When: Variance testing, goodness-of-fit, independence tests

Mean: μ = k, Variance: σ² = 2k

Parameter: k = degrees of freedom

Formula and tables will be PROVIDED

🔟 BETA DISTRIBUTION

When: Data on [0,1] interval, proportions, Bayesian analysis

Mean: μ = α/(α+β)

Flexible: Can be U-shaped, bell-shaped, skewed

Formula will be PROVIDED

1️⃣1️⃣ LOGNORMAL DISTRIBUTION

When: Positive data with right skew, ln(X) is normal

Key Property: If X ~ Lognormal, then ln(X) ~ Normal

Scenarios: Particle sizes, income, component lifetimes with wear

Formula will be PROVIDED

1️⃣2️⃣ WEIBULL DISTRIBUTION ⭐

When: Failure/reliability analysis, lifetime data, wear-out phenomena

Formula: f(x) = (α/β) × (x/β)(α-1) × exp(-(x/β)α)

Parameters: α = shape, β = scale

Special Cases: α=1 → Exponential; α>1 → wear-out; α<1 → infant mortality

Must Memorize? YES - CRITICAL for reliability

1️⃣3️⃣ UNIFORM DISTRIBUTION

When: All values equally likely over interval [a,b]

Formula: f(x) = 1/(b-a)

Mean: μ = (a+b)/2, Variance: σ² = (b-a)²/12

Distribution Selection Flowchart

DATA TYPE?
│
├─ DISCRETE (count)
│  ├─ Fixed trials? → Binomial (2 outcomes) or Multinomial
│  ├─ Events per unit? → Poisson
│  ├─ Until rth success? → Negative Binomial
│  ├─ Until 1st success? → Geometric
│  └─ No replacement? → Hypergeometric
│
└─ CONTINUOUS (measurement)
   ├─ Time/waiting?
   │  ├─ Until 1st event? → Exponential
   │  ├─ Until multiple events? → Gamma
   │  └─ Failure time? → Weibull
   ├─ Bell-shaped? → Normal
   ├─ Log-transformed normal? → Lognormal
   ├─ On [0,1]? → Beta
   ├─ Variance test? → Chi-square
   └─ Equal likelihood? → Uniform

Distribution Relationships

General Form → Special Case
Negative Binomial(r,p) → Geometric(p) when r=1
Gamma(α,β) → Exponential(λ) when α=1
Gamma(α,β) → Chi-square(k) when α=k/2, β=2
Binomial(n,p) → Normal(np, np(1-p)) as n→∞
Binomial(n,p) → Poisson(np) as n→∞, p→0
Hypergeometric → Binomial as N→∞

✏️ Practice Problems with Solution Strategies

Problem Type 1: Conditional Probability & Bayes' Theorem

Example Problem:

A manufacturing plant produces items with a 2% defect rate. An inspector catches 99% of defects but also incorrectly identifies 1% of good items as defective. If an item is flagged as defective, what is the probability it's actually defective?

Solution Strategy:

  1. Define events: D = actually defective, F = flagged as defective
  2. List given: P(D) = 0.02, P(F|D) = 0.99, P(F|Dc) = 0.01
  3. Find: P(D|F) using Bayes' theorem
  4. Calculate P(F): P(F) = P(F|D)P(D) + P(F|Dc)P(Dc) = 0.0198 + 0.0098 = 0.0296
  5. Apply Bayes: P(D|F) = (0.99 × 0.02) / 0.0296 ≈ 0.668

Answer: ~67% probability it's actually defective

Key Insight: Even with 99% accuracy, rare defects lead to high false positives!

Problem Type 2: Expected Value and Variance

Example Problem:

A company's profit X (in thousands) has distribution: Loss (-50) with p=0.2, Modest gain (20) with p=0.5, Large gain (100) with p=0.3. Calculate E(X) and Var(X).

Solution Strategy:

  1. Calculate E(X): Σ x·P(x) = (-50)(0.2) + (20)(0.5) + (100)(0.3) = 30
  2. Calculate E(X²): Σ x²·P(x) = 2500(0.2) + 400(0.5) + 10000(0.3) = 3700
  3. Calculate Var(X): E(X²) - [E(X)]² = 3700 - 900 = 2800
  4. SD(X): √2800 ≈ 52.9

Answer: E(X) = 30, Var(X) = 2800, SD(X) ≈ 52.9

Problem Type 3: Binomial Distribution

Example Problem:

A production line has 5% defect rate. In a sample of 20 items, what's the probability exactly 2 are defective?

Solution Strategy:

  1. Verify conditions: Fixed n=20, 2 outcomes, constant p=0.05 ✓
  2. Apply formula: P(X=2) = C(20,2) × (0.05)² × (0.95)¹⁸
  3. Calculate: C(20,2) = 190, result ≈ 0.179

Answer: ~17.9%

Problem Type 4: Poisson Distribution ⭐

Example Problem:

Manufacturing defects occur at 2 per 100 meters of cable. What's the probability of exactly 3 defects in 100 meters?

Solution Strategy:

  1. Identify: λ = 2 (average defects per 100m)
  2. Apply formula: P(X=3) = (e-2 × 23) / 3!
  3. Calculate: (0.1353 × 8) / 6 ≈ 0.180

Answer: ~18.0%

Key: Mean = Variance = 2

Problem Type 5: Normal Distribution & Z-Scores

Example Problem:

Component lifetimes are Normal(μ=1000, σ=100). What's P(900 < X < 1100)?

Solution Strategy:

  1. Standardize: Z₁ = (900-1000)/100 = -1.00, Z₂ = (1100-1000)/100 = 1.00
  2. Find: P(-1 < Z < 1) from Z-table = 0.8413 - 0.1587 = 0.6826
  3. Alternative: Use empirical rule - this is ±1σ from mean = 68%

Answer: ~68.26%

Problem Type 6: Finding Critical Values

Example Problem:

Component lifetimes are Normal(1000, 100). Find the lifetime below which 10% of components fail.

Solution Strategy:

  1. Find: z₀ where P(Z < z₀) = 0.10 from Z-table → z₀ ≈ -1.28
  2. Convert back: x₀ = μ + z₀·σ = 1000 + (-1.28)(100) = 872

Answer: 872 hours (10th percentile)

Problem Type 7: Exponential & Weibull

Example Problem:

Component failures follow Exponential with mean 5000 hours. a) P(fail within 1000 hours)? b) Given working after 3000 hours, P(works another 1000)?

Solution Strategy:

  1. Part a: λ = 1/5000, P(X ≤ 1000) = 1 - e-1000/5000 = 1 - e-0.2 ≈ 0.1813
  2. Part b: Use memoryless: P(X > 1000) = e-0.2 ≈ 0.8187

Answers: a) 18.13%, b) 81.87%

Key: Memoryless property - past doesn't matter!

Problem Type 8: Linear Regression

Example Problem:

Data: (2,50), (4,65), (6,75), (8,85), (10,95). Find regression equation ŷ = b₀ + b₁x

Solution Strategy:

  1. Calculate: Σx=30, Σy=370, Σxy=2440, Σx²=220, n=5
  2. Find b₁: b₁ = [2440 - 5(6)(74)] / [220 - 5(36)] = 220/40 = 5.5
  3. Find b₀: b₀ = 74 - 5.5(6) = 41
  4. Equation: ŷ = 41 + 5.5x

Interpretation: Each hour of study increases score by 5.5 points

Problem Type 9: Concept Integration

Example Problem:

Two independent quality checks (95% and 90% sensitive). Defect rate is 3%. a) P(both miss a defect)? b) P(defective | passes both)?

Solution Strategy (Part a):

P(both miss | defect) = (1-0.95)(1-0.90) = 0.05 × 0.10 = 0.005

Solution Strategy (Part b):

Use Bayes' theorem with above calculation

Answer: ~0.155% (very low - system is effective!)

Common Exam Mistakes to Avoid

  • ✗ Using binomial when hypergeometric needed (check: with or without replacement?)
  • ✗ Confusing P(A|B) with P(B|A)
  • ✗ Forgetting variance scales by a² for Var(aX)
  • ✗ Not standardizing before using Z-tables
  • ✗ Using wrong regression formula order

⭐ Key Formulas & Memorization Guide

⭐⭐⭐ ABSOLUTE MEMORIZATION REQUIREMENTS

1. Normal Distribution (MUST MEMORIZE)

📐 PDF Formula:
f(x) = 1σ√(2π) × e−(x−μ)²/2σ²
Where μ is mean, σ is standard deviation, e ≈ 2.71828
🔄 Standardization (Z-score):
Z = (X − μ) / σ
Convert any normal variable to standard normal (Z-table lookup)
📊 Empirical Rule:
• 68% of data within μ ± σ
• 95% of data within μ ± 2σ
• 99.7% of data within μ ± 3σ

2. Bayes' Theorem (MUST MEMORIZE)

🎯 Main Formula:
P(A | B) = P(B | A) × P(A)P(B)
Find probability of cause A given evidence B
📈 Extended Form (Law of Total Probability):
P(A | B) = P(B | A) × P(A)P(B | A) × P(A) + P(B | Ac) × P(Ac)
Use when you need to calculate P(B) from all possible causes

3. Poisson Distribution (MUST MEMORIZE)

📊 Probability Mass Function:
P(X = x) = e−λ × λxx!
Where λ is the average rate and x is the number of events
📈 Mean & Variance:
Mean: μ = λ
Variance: σ2 = λ
⭐ CRITICAL: Mean equals Variance for Poisson!

4. Exponential Distribution (MUST MEMORIZE)

📊 Probability Density Function:
f(x) = λe−λx, where x ≥ 0
CDF: F(x) = 1 − e−λx
📈 Mean & Variance:
Mean: μ = 1/λ
Variance: σ2 = 1/λ2
🔄 Memoryless Property (CRITICAL):
P(X > s + t | X > s) = P(X > t)
Past time doesn't affect future probability

5. Weibull Distribution (MUST MEMORIZE)

📊 Probability Density Function:
f(x) = αβ × xβα−1 × e−(x/β)α, where x ≥ 0
CDF: F(x) = 1 − e−(x/β)α
⚙️ Parameters:
• α (alpha) = Shape parameter
• β (beta) = Scale parameter (characteristic life)

6. Binomial Distribution (MUST MEMORIZE)

📊 Probability Mass Function:
P(X = x) = C(n,x) × px × (1−p)(n−x)
Where C(n,x) = n!x!(n−x)!
📈 Mean & Variance:
Mean: μ = np
Variance: σ2 = np(1−p)

7. Conditional Probability (MUST MEMORIZE)

🎯 Basic Formula:
P(A | B) = P(A ∩ B)P(B)
Probability of A given B has occurred
🔀 Product Rule:
P(A ∩ B) = P(A | B) × P(B) = P(B | A) × P(A)
Joint probability equals conditional × individual probability
✓ Independence Condition:
P(A ∩ B) = P(A) × P(B)
If and only if A and B are independent

8. Expected Value (MUST KNOW)

📊 Discrete Variables:
E(X) = Σ [x · P(x)]
Sum of each value times its probability
📊 Continuous Variables:
E(X) = ∫ [x · f(x)] dx
Integral of value times its probability density
🔄 Linear Function:
E(aX + b) = aE(X) + b
Linearity of expectation property

9. Variance (MUST KNOW)

📐 Definition:
Var(X) = E(X²) − [E(X)]²
Expected squared deviation from the mean
🔄 Linear Function:
Var(aX + b) = a² · Var(X)
⚠️ Note: Variance scales by a², not a!
➕ Sum of Independent Variables:
Var(X + Y) = Var(X) + Var(Y)
Only true if X and Y are independent

10. Least Squares Regression (MUST MEMORIZE)

📚 Understanding SST, SSR, and SSE

SST = Total Sum of Squares
SST = Σ(yᵢ − ȳ)²
Measures total variation in your data (how much data varies from the mean)
SSR = Regression Sum of Squares
SSR = Σ(ŷᵢ − ȳ)²
Measures variation EXPLAINED by your regression model (good part)
SSE = Error Sum of Squares (Residual)
SSE = Σ(yᵢ − ŷᵢ)²
Measures variation NOT explained by regression (bad part - errors)
📊 Key Relationship
SST = SSR + SSE
Total variation = Explained variation + Unexplained variation
📐 Regression Coefficients:
b₁ = Σxy − n(x̄)(ȳ)Σx² − n(x̄)²
Slope of the regression line
b₀ = ȳ − b₁x̄
Y-intercept of the regression line
📊 Fitted Regression Line:
ŷ = b₀ + b₁x
📈 Residuals & Sum of Squares:
Residual: eᵢ = yᵢ − ŷᵢ (actual − predicted)
SST = Σ(yᵢ − ȳ)² (Total variation)
SSR = Σ(ŷᵢ − ȳ)² (Explained by regression)
SSE = Σ(yᵢ − ŷᵢ)² (Unexplained error)

11. Normal Approximation to Binomial

✓ Conditions Required:
• np ≥ 10
• n(1−p) ≥ 10
Both conditions must be met for valid approximation
📊 Approximation:
If X ~ Binomial(n,p), then approximately
X ~ Normal(μ = np, σ2 = np(1−p))
🔄 Continuity Correction:
P(X = k) ≈ P(k − 0.5 < X < k + 0.5)
Adjust for discrete-to-continuous conversion

1️⃣ NORMAL DISTRIBUTION FORMULA

📍 From: Chapter 6, Section 6.2

The Formula: f(x) = 1σ√(2π) × e−(x−μ)²/2σ²

✅ When to Use It:

  • Measuring continuous data that appears bell-shaped
  • Natural phenomena (heights, weights, test scores, blood pressure)
  • Quality control (product measurements within tolerance)
  • Manufacturing processes (machine output variation)
  • Finding probabilities for natural measurements

❌ When NOT to Use It:

  • Don't use for discrete count data (use Poisson instead)
  • Don't use when data is skewed (use Lognormal instead)
  • Don't use for bounded data like percentages (use Beta instead)

🔍 Real-World Example:

Scenario: A battery manufacturer produces batteries with voltage that follows a normal distribution with mean 12V and standard deviation 0.5V. What percentage of batteries have voltage between 11.5V and 12.5V?

📊 Solution:

Step 1: Recognize the problem type
This is a normal distribution problem - continuous measurement with bell-shaped data
Step 2: Convert to Z-scores
Z₁ = (11.5 - 12) / 0.5 = −1.0
Z₂ = (12.5 - 12) / 0.5 = 1.0
Step 3: Use Z-table or empirical rule
P(−1 < Z < 1) = 68% (empirical rule: within ±1σ of mean)
✅ Answer
68% of batteries have voltage between 11.5V and 12.5V

  • Z-score standardization (Chapter 6, Section 6.3)
  • Empirical Rule (Chapter 6)
  • Normal Approximation to Binomial (Chapter 6, Section 6.5)

2️⃣ BAYES' THEOREM FORMULA

📍 From: Chapter 2, Section 2.7

The Formula: P(A|B) = P(B|A) × P(A)P(B)

✅ When to Use It:

  • Medical testing (given positive test result, what's probability of disease?)
  • Quality control (given defect detected, what's probability it's real?)
  • Spam detection (given spam-like email, what's probability it's spam?)
  • Finding posterior probability (cause given evidence)
  • Classification problems (determining category given observations)

❌ When NOT to Use It:

  • Don't use for simple probability (just use multiplication rule)
  • Don't use when events are independent (P(A|B) = P(A) anyway)
  • Don't confuse P(A|B) with P(B|A) - they're different!

🔍 Real-World Example:

Scenario: A disease screening test has these characteristics:

  • Disease prevalence: 1% of population
  • Test sensitivity: 95% (catches 95% of actual cases)
  • False positive rate: 5% (5% of healthy people test positive)

Question: If someone tests positive, what's the probability they actually have the disease?

📊 Solution:

Step 1: Define events
D = person has disease, P(D) = 0.01
T+ = test is positive
P(T+|D) = 0.95 (sensitivity)
P(T+|Dc) = 0.05 (false positive rate)
Step 2: Calculate P(T+)
P(T+) = P(T+|D)×P(D) + P(T+|Dc)×P(Dc)
= (0.95 × 0.01) + (0.05 × 0.99) = 0.0095 + 0.0495 = 0.059
Step 3: Apply Bayes' Theorem
P(D|T+) = (0.95 × 0.01) / 0.059 = 0.0095 / 0.059 ≈ 0.161
✅ Answer
Only 16.1% probability person actually has disease despite positive test!
This is why doctors recommend confirmatory tests before diagnosis

📚 Related Concepts:

  • Conditional Probability (Chapter 2, Section 2.5)
  • Total Probability Law (Chapter 2)
  • Independence (Chapter 2, Section 2.5)

3️⃣ POISSON DISTRIBUTION FORMULA

📍 From: Chapter 5, Section 5.5

The Formula: P(X = x) = e−λ × λxx!

Key Property: Mean = λ, Variance = λ

✅ When to Use It:

  • Rare events in fixed intervals
  • Arrivals/departures (customers per hour, calls per minute)
  • Defects per unit (defects per 100 meters of cable)
  • Accidents (traffic accidents per day)
  • Biological events (mutations per generation)

❌ When NOT to Use It:

  • Don't use for fixed number of trials (use Binomial instead)
  • Don't use when rate changes over interval
  • Don't use if events affect each other (violates independence)

🔍 Real-World Example:

Scenario: A call center receives an average of 3 calls per minute. What's the probability of getting exactly 5 calls in the next minute?

📊 Solution:

Step 1: Verify Poisson conditions
✓ Events (calls) occur independently
✓ Constant average rate (3 per minute)
✓ Counting events in fixed interval (1 minute)
Step 2: Identify parameter
λ = 3 calls per minute
Step 3: Apply formula
P(X = 5) = (e−3 × 35) / 5!
= (0.0498 × 243) / 120 ≈ 0.101
✅ Answer
10.1% probability of exactly 5 calls in next minute

🎯 Key Insight:

Mean = Variance for Poisson! Both equal λ. This property helps identify Poisson scenarios.

📚 Related Concepts:

  • Connection to Exponential Distribution (Chapter 6, Section 6.7)
  • Poisson Process (Chapter 5, Section 5.5)
  • Normal Approximation (Chapter 6, Section 6.5)

4️⃣ EXPONENTIAL DISTRIBUTION FORMULA

📍 From: Chapter 6, Section 6.7

The Formula: f(x) = λe−λx, where x ≥ 0

Key Property (Memoryless): P(X > s + t | X > s) = P(X > t)

✅ When to Use It:

  • Time between events in Poisson process
  • Waiting times (time until next customer)
  • Equipment lifetime (no wear, random failure)
  • Service times (how long service takes)
  • Interarrival times (time between arrivals)

⭐ CRITICAL PROPERTY - Memoryless:

P(X > s + t | X > s) = P(X > t)
Past time doesn't matter - system "forgets" how long you've waited!

❌ When NOT to Use It:

  • Don't use for time with wear/aging (use Weibull instead)
  • Don't use for multiple events until kth event (use Gamma instead)

🔍 Real-World Example:

Scenario: Average time between customer arrivals at a store is 10 minutes.

Question a: What's probability customer arrives within next 5 minutes?
Question b: Given no customer for 10 minutes, what's probability next customer arrives within 5 minutes?

📊 Solution:

Step 1: Identify parameter
Mean = 10 minutes, so λ = 1/10
Step 2: Part (a) - Basic probability
P(X ≤ 5) = 1 − e−(1/10)×5 = 1 − e−0.5
= 0.393 or 39.3%
Step 3: Part (b) - Using memoryless property
P(X ≤ 5 | X > 10) = P(X ≤ 5) = 0.393 or 39.3%
Same answer! Past waiting doesn't help predict future
✅ Key Insight
The "memoryless" property makes exponential special - previous experience is irrelevant!

📚 Related Concepts:

  • Connection to Poisson Process (Chapter 5)
  • Gamma Distribution (Chapter 6, Section 6.6)
  • Memoryless Property (Chapter 6, Section 6.7)

5️⃣ WEIBULL DISTRIBUTION FORMULA

📍 From: Chapter 6, Section 6.11

The Formula: f(x) = αβ × xβα−1 × e−(x/β)α

✅ When to Use It:

  • Reliability engineering (equipment failure times)
  • Component lifetime (electronic components, bearings)
  • Failure analysis (predicting when items will fail)
  • Warranty periods (how long until failure)
  • Strength of materials (material breaking strength)

🔑 Key Parameter - Shape (α):

  • α < 1: Decreasing failure rate (infant mortality - early failures)
  • α = 1: Constant failure rate (exponential case)
  • α > 1: Increasing failure rate (wear-out - failures increase with time)

❌ When NOT to Use It:

  • Don't use for simple arrivals (use Poisson/Exponential instead)
  • Don't use when aging effects are negligible

🔍 Real-World Example:

Scenario: Electronic component lifetime follows Weibull distribution with:

  • Shape parameter: α = 2.5 (wear-out mode)
  • Scale parameter: β = 5000 hours

Question: What's probability component fails before 3000 hours?

📊 Solution:

Step 1: Recognize Weibull problem
✓ Reliability/failure time analysis
✓ α > 1 (wear-out scenario)
Step 2: Use CDF formula
F(3000) = 1 − e−(3000/5000)2.5
= 1 − e−(0.6)2.5
= 1 − e−0.262 ≈ 0.231
✅ Answer
23.1% probability component fails before 3000 hours

📚 Related Concepts:

  • Special case: Exponential (α = 1)
  • Gamma Distribution relationship
  • Reliability engineering (Chapter 6)

6️⃣ BINOMIAL DISTRIBUTION FORMULA

📍 From: Chapter 5, Section 5.2

The Formula: P(X = x) = C(n,x) × px × (1−p)(n−x)

Where: Mean = np, Variance = np(1−p)

✅ When to Use It:

  • Fixed number of trials (n fixed in advance)
  • Two outcomes per trial (success/failure, yes/no)
  • Independent trials (one trial doesn't affect another)
  • Constant probability (p same for all trials)
  • Quality control, surveys, product testing

❌ When NOT to Use It:

  • Don't use if sampling without replacement (use Hypergeometric instead)
  • Don't use for variable number of trials (use Negative Binomial)
  • Don't use if probability changes between trials

🔍 Real-World Example:

Scenario: A manufacturing process produces 95% defect-free items (5% defective). A sample of 20 items is taken. What's probability exactly 2 items are defective?

📊 Solution:

Step 1: Verify Binomial conditions
✓ Fixed n = 20 trials
✓ Two outcomes: defective (p = 0.05) or good (0.95)
✓ Independent items, constant p
Step 2: Calculate C(20,2)
C(20,2) = 20!/(2!×18!) = 190
Step 3: Apply formula
P(X = 2) = 190 × (0.05)2 × (0.95)18
= 190 × 0.0025 × 0.3585 ≈ 0.189
✅ Answer
18.9% probability exactly 2 out of 20 are defective

  • Hypergeometric vs. Binomial (Chapter 5, Section 5.3)
  • Normal Approximation to Binomial (Chapter 6, Section 6.5)
  • Multinomial Distribution (Chapter 5, Section 5.2)

7️⃣ CONDITIONAL PROBABILITY FORMULA

📍 From: Chapter 2, Section 2.5

The Formula: P(A|B) = P(A ∩ B)P(B)

Product Rule: P(A ∩ B) = P(A|B) × P(B)

✅ When to Use It:

  • Given that something happened, find probability of something else
  • Sequential events (first event affects probability of second)
  • Dependent events (probability changes based on prior information)
  • Risk assessment (probability given risk factors)

❌ When NOT to Use It:

  • Don't confuse P(A|B) with P(B|A) - they're different!
  • Don't use for independent events where P(A|B) = P(A)

🔍 Real-World Example:

Scenario: Weather prediction with two connected events:

  • P(Rain tomorrow) = 0.3
  • P(Traffic jam | Rain) = 0.8
  • P(Traffic jam | No rain) = 0.1

Question: What's probability of both rain AND traffic jam?

📊 Solution:

Step 1: Recognize Conditional Probability problem
✓ Two events connected (rain affects traffic)
✓ Given one event, find probability of outcome
Step 2: Use Product Rule
P(Rain AND Traffic) = P(Traffic|Rain) × P(Rain)
= 0.8 × 0.3 = 0.24
✅ Answer
24% probability of both rain and traffic jam
📌 Insight
Traffic depends on rain, so we multiply conditional probability by base probability

  • Bayes' Theorem (Chapter 2, Section 2.7)
  • Total Probability Law (Chapter 2)

8️⃣ EXPECTED VALUE & VARIANCE FORMULAS

📍 From: Chapter 4, Sections 4.1-4.2

The Formulas:

E(X) = Σ[x × P(x)] for discrete

Var(X) = E(X²) − [E(X)]²

Linear Transformation: E(aX + b) = aE(X) + b, Var(aX + b) = a² × Var(X)

✅ When to Use It:

  • Finding average outcome (expected value)
  • Measuring spread/risk (variance)
  • Comparing investments (which has higher return, lower risk?)
  • Linear transformations (scaling, unit changes)
  • Decision making (choose option with highest expected value)

❌ When NOT to Use It:

  • Don't forget Var(aX) = a² Var(X), not a Var(X)
  • Don't add variances of dependent variables

🔍 Real-World Example:

Scenario: Investment decision with uncertain returns.

Stock A: Gain $100 with probability 0.6, Loss $50 with probability 0.4

Question: Should you invest? Calculate expected value and risk.

📊 Solution:

Step 1: Calculate E(X)
E(X) = 100(0.6) + (−50)(0.4) = 60 − 20 = $40 expected gain
Step 2: Calculate E(X²)
E(X²) = (100)²(0.6) + (−50)²(0.4) = 6000 + 1000 = 7000
Step 3: Calculate Var(X)
Var(X) = 7000 − (40)² = 7000 − 1600 = 5400
SD(X) = √5400 ≈ $73.48 risk
✅ Answer
Expected gain: $40, Risk (std dev): $73.48
High risk for moderate reward - depends on risk tolerance!

  • Chebyshev's Theorem (Chapter 4, Section 4.4)
  • Properties of Expectation (Chapter 4)

9️⃣ LEAST SQUARES REGRESSION FORMULAS

📍 From: Chapter 11, Sections 11.2-11.3

The Formulas:

b₁ = Σxy − n(x̄)(ȳ)Σx² − n(x̄)², b₀ = ȳ − b₁x̄, ŷ = b₀ + b₁x

✅ When to Use It:

  • Predicting one variable from another (continuous data)
  • Finding relationship between variables
  • Trend analysis (sales over time, growth patterns)
  • Cost estimation (predicting costs from units produced)
  • Scientific studies (relationship between measured variables)

❌ When NOT to Use It:

  • Don't extrapolate far beyond data range (unreliable)
  • Don't use for non-linear relationships without transformation
  • Don't assume causation just because correlation exists

🔍 Real-World Example:

Scenario: Relationship between study hours and exam scores.

Data: 5 students with study hours: 2, 4, 6, 8, 10 and scores: 50, 65, 75, 85, 95

Question: Find regression equation and predict score for 7 hours study.

📊 Solution:

Step 1: Calculate necessary sums
Σx = 30, Σy = 370, Σxy = 2440, Σx² = 220, n = 5
x̄ = 6, ȳ = 74
Step 2: Calculate b₁
b₁ = [2440 − 5(6)(74)] / [220 − 5(36)]
= [2440 − 2220] / [220 − 180] = 220 / 40 = 5.5
Step 3: Calculate b₀
b₀ = 74 − 5.5(6) = 74 − 33 = 41
Step 4: Write regression equation
ŷ = 41 + 5.5x
Step 5: Prediction for 7 hours
ŷ = 41 + 5.5(7) = 41 + 38.5 = 79.5 score
✅ Interpretation
Each additional study hour increases expected score by 5.5 points
Base score (with 0 hours) would be 41
